# All FishingTackle Shop features

### All actors
- Guest
- User
- Admin
- Super Admin

### A. Admin
- Order management: View orders, order processing (approve, delete)
- Customer management
- Product management
- Order statistics
- Product statistics

### B. Super Admin
- All admin features
- Manufacturer management
- Staff management

### C. Guest
- Live search
- Add to cart, view cart
- Sign up, sign in, forgot password

### D. User
- All guest features
- Rating & review
- Checkout
- Edit personal information
- View payment history
